# 🎮 I Wanna Be The Flixel
### An "I Wanna Be The Guy" fangame created with [HaxeFlixel](https://haxeflixel.com)

Created from the ground up, this is an individual project of recreating a difficult and fun to play platformer.

 📝 My goal is creating 5 full chapters (worlds) and **AT LEAST** 6 bosses
